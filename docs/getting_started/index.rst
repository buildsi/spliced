===============
Getting Started
===============

Spliced is software for performing or emulating splices, meaning subbing in one version of a library
for another, and predicting (through many avenues) whether it will work or not.

If you have any questions or issues, please `let us know <https://github.com/buildsi/spliced/issues>`_
